# Gerador de Senhas Seguras em Python

Projeto em Python para gerar senhas seguras utilizando o módulo `secrets`

## Funcionalidades
- Geração de senhas:
  - Hexadecimal
  - Bytes
  - Caracteres (letras, números e símbolos)
- Interface via terminal
- Validação de entradas do usuário

## Tecnologias
- Python 3
- secrets
- string

## Como executar
```bash
python main.py
